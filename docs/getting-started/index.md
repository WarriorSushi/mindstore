# Getting Started

This section helps new users go from zero to a useful MindStore setup quickly.

## What You Will Learn

- How MindStore stores your knowledge.
- How to import your first source.
- How to search, chat, and connect an AI client.
- How to decide whether to run local, self-hosted, or hosted-ready.

## Recommended Path

1. Read [Concepts](../concepts/index.md) to understand the product model.
2. Follow the [quickstart](./quickstart.md).
3. Use the [first-run checklist](./first-run-checklist.md) to verify the install is actually healthy.
4. Read [Provider Access](../build/provider-access.md) if you want to understand what is required versus optional.
5. Set up [MindStore Everywhere](./mindstore-everywhere.md) if you want browser capture.
6. Try one [import guide](../import-guides/index.md).
7. Connect MCP using [MCP](../mcp/index.md).
