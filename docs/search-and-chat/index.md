# Search and Chat

Search and chat are the two ways users get value after import.

## Search

- Search should be fast, explainable, and useful even before AI is configured.
- Results should show source, date, and enough context to trust the match.

## Chat

- Chat should assemble relevant memories as context.
- Answers should cite the memories they used.
- When AI providers are unavailable, the app should still guide the user through import, browsing, and setup.
