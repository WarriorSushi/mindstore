# Deploy

This section should help users choose the simplest deployment that still fits their use case.

## Deployment Modes

- Local developer mode.
- Self-hosted single-user mode.
- Self-hosted small-team mode.
- Hosted-ready mode for future scaling.

See [Deployment Modes](./deployment-modes.md) for the current target model.
