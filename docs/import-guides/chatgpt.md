# ChatGPT Export

Use this guide when you have a ChatGPT export ZIP or JSON file.

## What It Does

- Parses conversations into searchable memories.
- Preserves conversation structure where possible.
- Keeps the data locally in your MindStore database.

## How To Verify

- The import finishes without errors.
- The source appears in the import history.
- Search can find phrases from the uploaded conversations.
