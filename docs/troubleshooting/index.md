# Troubleshooting

This section should help users recover quickly when something does not work.

## Common Problems

- Database is not connected.
- AI provider is not configured.
- Import fails because the source format is invalid.
- MCP client cannot connect.
- Search returns empty or irrelevant results.

## Rule Of Thumb

Every troubleshooting page should end with a fast verification step so users know when the issue is fixed.
