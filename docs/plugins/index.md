# Plugins

Plugins are how MindStore becomes a platform instead of a single app.

## Plugin Philosophy

- Built-ins can ship with the repo.
- Community plugins should be installable, documented, and safely extensible.
- Plugin contracts should be stable, explicit, and easy to validate.

## What A Plugin Should Define

- Manifest metadata.
- Capabilities and permissions.
- UI surfaces such as settings, widgets, or pages.
- Hooks and jobs.
- MCP tools or resources when relevant.
