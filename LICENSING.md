## The 2-Year Conversion

Each version of MindStore automatically converts to the **MIT License** two years after its release date. This means:

- Code released today → becomes MIT on this date in 2028
- Code released next month → becomes MIT one month later
- The latest version always has FSL protection

This is a rolling window. Our most recent code is always protected. Older versions gradually become fully open source.

## Why FSL?

We want MindStore to be genuinely useful to everyone — self-hosters, students, companies, researchers. The only thing we restrict is taking our code and offering it as a competing hosted service. That's how we sustain development.

## Plugin Licensing

Plugins built using the MindStore Plugin SDK may be licensed under any terms of the author's choosing. Building a plugin does not create a derivative work of MindStore.
